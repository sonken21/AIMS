# Setup/run project

###### Clone the repository

```sh
git clone https://github.com/TanLP2003/TKXDPM.20241.19.git
```

```sh
cd TKXDPM.20241.19
```

###### Install needed dependency:

```sh
./mvnw.cmd clean install
```

###### Run project:

```sh
./mvnw.cmd javafx:run
```

###### Prerequisites

- Java 21 or higher

- Apache Maven 3.13.x or higher

- JDK 21 or later

- IntelliJ IDEA
